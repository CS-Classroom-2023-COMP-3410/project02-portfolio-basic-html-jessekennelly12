Summary

This is a small 2D platformer concept featuring a cat character, bounce mechanics, and a “victory goal” object at the end of each level.

“If the cat can’t double-jump, is it even a platformer?”

Features

  • Bouncy movement mechanic (timing matters)
  • Collectible items (for fun + feedback)
  • Simple level goal trigger (win animation idea)
  • Iterative debugging process (my true final boss)
